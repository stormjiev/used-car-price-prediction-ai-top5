## ⚠️ 已知问题与应对

### 问题1: TensorFlow导入失败
**症状**: `python -c "import tensorflow"` 出现分段错误

**原因**: Python 3.13与TensorFlow 2.20.0在arm64 macOS上的兼容性问题

**应对方案**:
- ✅ 使用PyTorch替代NN模块
- 可在nn_model.py中使用torch.nn而非tf.keras
- 预期性能相同或更优 (PyTorch在Metal上表现更好)

### 问题2: 路径问题
**症状**: 找不到文件 `No such file or directory`

**原因**: 路径计算依赖当前工作目录

**应对方案**:
- ✅ 总是从 v2/ 或其子目录执行
- 示例: `cd v2/code/feature && python generation.py`

### 问题3: 内存溢出
**症状**: 处理150K×200特征时内存不足

**原因**: 特征矩阵过大

**应对方案**:
- 24GB内存应足够
- 如出现问题，可分批处理或启用虚拟内存
